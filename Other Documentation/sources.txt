============================================================

Library Management System

============================================================

Developed Using: Python (PyQt6, SQLite, Google Drive API, Google Sheets API)

============================================================

Description:
A desktop application built with Python and PyQt6 for managing library records, syncing data with Google Drive, and exporting to Google Sheets.
